**Debezium** — это open-source платформа для **Change Data Capture (CDC)**, т.е. отслеживания изменений в базе данных в реальном времени и трансляции этих изменений в другие системы (обычно через Apache Kafka). 

Debezium совместим с большим количеством популярных БД, таких как: MySQL, PostgreSQL, MongoDB, MSSQL, Oracle, Cassandra.

Debezium отслеживает изменения в базе данных с помощью встроенных механизмов репликации, предоставляемых самой СУБД, например, читает журнал изменений базы данных.

Debezium используют для:
- построения Event-Driven архитектуры,
- наполнения кэша,
- синхронизации данных между системами.
