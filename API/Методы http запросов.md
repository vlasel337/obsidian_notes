HTTP методы:
- Метод PUT используется для создания или **полного** пересоздания ресурса.
Основное свойство метода PUT – [[Идемпотентность|идемпотентность]] (то есть повторные вызовы метода приводят к одному и тому же состоянию ресурса).
- Метод PATCH используется для частичного изменения ресурса.
