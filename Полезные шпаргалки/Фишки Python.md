#### 3) Побитовая инверсия, оператор ~
В Python оператор ~ рядом с числом выполняет побитовую инверсию, то есть выполняет операцию для x: ~x = -(x + 1)
```
numbers = [1, 2, 3, 4, 5]
numbers[~0] # -> numbers[-1] -> 5
numbers[~1] # -> numbers[-2] -> 4
```
